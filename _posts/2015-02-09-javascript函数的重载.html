javascript函数不存在重载的概念。函数的个数与调用没有关系，后面会覆盖前面的定义。 

function sum(num1,num2){
	return num1+num2;
}


function sum(num1){
	return num1+100;
}

alert(sum(5));
alert(sum(5,10));

结果都是105，即调用最后一个sum函数。

换一种方法，相信大家会容易明白点。 

var sum=function(num1,num2){
	return num1+num2;
};

var sum=function(num1){
	return num1+100;
};

alert(sum(5));
alert(sum(5,10));
