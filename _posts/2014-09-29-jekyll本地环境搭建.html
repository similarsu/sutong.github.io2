注册github账号

安装ruby

1、下载railsinstaller点此下载

2、安装railsinstaller

配置SSH

1、生成SSH

2、登陆github网站，并添加SSH

安装jekyll

1、打开命令行窗口

2、测试一下git是否连接正常 

ssh -T git@github.com

3、防止出错 

set LC_ALL=en_US.UTF-8
set LANG=en_US.UTF-8

4、稍微对配置做一下修改，把淘宝的镜像加到gem的镜像列表里 

gem sources --remove http://rubygems.org/
gem sources -a http://ruby.taobao.org/

5、查看 

gem sources -l

6、安装jekyll 

gem install jekyll

7、是否安装成功jekyll 

jekyll -v

8、安装rdiscount kramdown 

gem install rdiscount kramdown

9、新建jekyll项目 

jekyll new blog

10、进入并运行jekyll项目 

cd blog
jekyll serve

11、打开浏览器，输入http://127.0.0.1:4000
