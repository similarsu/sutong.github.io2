javascript函数是一种特殊的对象。

函数在内存中是以键值对的形式存储的。

函数的定义方式有如下三种。

第一种方式 

function fn1(){
		alert("fn1");
	}

第二种方式 

var fn2=function(){
	alert("fn2");
}

第三种方式(不常见) 

var fn3=new Function("num1","num2","alert(num1+num2)");
fn3(1,2);
