javascript函数是一种特殊的对象。

函数的赋值是通过拷贝实现的。 

var fn1=function(){
	alert("fn1");
}
var fn2=fn1;
fn1();
fn2();
fn1=function(){
	alert("fn11");
}
fn1();
fn2();

对象的赋值是通过改变引用的指向实现的。 

var a=new Object();
b=a;
a.name="Jone";
alert(b.name);
