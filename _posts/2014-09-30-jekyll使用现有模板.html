1、打开命令行窗口

2、使用现有的jekyll模版 

git clone git@github.com:sutong/sutong.github.io.git sutong.github.io

3、进入目录 

cd sutong.github.io

4、删除.git文件 

rm -rf .git

5、初始化git 

git init

6、添加文件到版本控制 

git add -A

7、提交到本地库 

git commit -m "init the git"

8、添加中央仓库地址 

git remote add origin git@github.com:sutong/sutong.github.io.git

9、提交到中央仓库 

git push -u origin master
