1、安装epel源 


yum install http://dl.fedoraproject.org/pub/epel/6/x86_64/epel-release-6-8.noarch.rpm

2、安装docker 


yum install docker-io

3、启动docker 


service docker start
chkconfig docker on#设置开机启动

4、验证是否启动成功 


docker info
